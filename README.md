# Personal Website

A minimalist, black-and-white personal website showcasing engineering projects.

## Features

- **Minimalist Design**: Apple Store-inspired aesthetic with strict black and white color scheme
- **Two Project Sections**: Separate sections for Mechanical Engineering and Software Development projects
- **Interactive Elements**: Smooth animations and hover effects throughout
- **Responsive**: Fully responsive design for all screen sizes

## Getting Started

### Installation

```bash
npm install
```

### Running the Development Server

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Building for Production

```bash
npm run build
```

## Customization

- Update project information in `src/components/Projects.js`
- Modify personal information in `src/components/Hero.js`
- Update contact links in `src/components/Contact.js`
